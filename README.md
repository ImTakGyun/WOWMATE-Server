# WOWMATE 서버 레포지토리
---

### 개발환경
- 프레임워크: spring boot 2.7.7
- build: gradle
- 언어: java 11
<br>

---

### 주의사항
- commit, push 하기 전에 pull 하기
- 기능 구현의 브랜치명은'feat/구현한기능'으로 통일 (따옴표 빼기)
- 작업한 브랜치에서 develop 브랜치로 push하고 PR 하기(checkout으로 브랜치 확인 꼭 하기)
- 자신이 해야할 일 Issues에 기록하기
- PR에서 관련된 Issues 번호 표시하기
- merge 전 다른 팀원 확인 필수
